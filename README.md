# Value Curve Studio

An interactive single-page value curve tool for comparing a product against competitors, scoring value points, and shaping a future strategic curve.

## Open Locally

Open `index.html` in any modern browser. The app is self-contained and does not require a build step.

## Features

- Add, reorder, and annotate value points
- Score current product and competitor curves by dragging chart points
- Switch to future strategy mode and classify value points as keep, eliminate, reduce, raise, or create
- Import and export project data as JSON

## Source

The app is packaged as a standalone HTML file generated from the original `价值曲线工具_v6.html`.
